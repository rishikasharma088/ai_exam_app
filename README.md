# AI Exam Preparation App

A simple Flutter based AI Exam Preparation App that allows users to practice multiple-choice questions.

## Features

- User Login and Signup
- Multiple Choice Questions
- Timer for each question
- Instant Correct / Wrong feedback
- Score calculation
- Beautiful result screen

## Technologies Used

- Flutter
- Dart
- Material UI

## How to Run

1. Clone the repository
2. Run the following commands

flutter pub get
flutter run
